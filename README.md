## 使用方法
```console:console
$ https://github.com/piyoraik/nest-backend
$ cd nest-backend
$ make up
$ make migration.sync
```
>### ※注意※
>クローンしたあとに`.env.sample`のファイル名を`.env`に修正し  
>`JWT_SECRET_KEY`の値を設定する

### ドキュメントURL
```
http://localhost:9000/api
```
